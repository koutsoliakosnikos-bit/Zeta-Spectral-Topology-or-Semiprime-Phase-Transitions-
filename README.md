Overview

Traditional approaches to algorithmic complexity rely on pure algebra. This framework hypothesizes that cryptographic hardness can be evaluated heuristically by routing topological and relativistic physics proxies through a state-of-the-art Deep Learning architecture.

Explicit 2D Geometry (GNN): To enforce local mathematical constraints.
Maxwell's Demon Gating: A sparse routing agent that acts as a dynamic filter, dropping "noisy" pairwise mathematics into a residual trash memory while preserving the core embedding.
